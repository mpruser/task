# README
## 로컬 실행([DEMO](https://mpruser.github.io/task/))

### 1. 카카오 API Key 설정
`${root}/.env` 파일의 카카오 API KEY 입력이 반드시 필요합니다.
```shell
KAKAO_API_KEY="YOUR_API_KEY"
```

### 2. 설치
```shell
$ yarn install --frozen-lockfile // or npm install --package-lock-only
```

### 3. 실행
```shell
$ yarn run // or npm run start
```
